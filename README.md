# Pomodoro Timer

Timer Pomodoro interativo feito com HTML, CSS e JavaScript puro.

## Funcionalidades

- Ciclos de foco (25 min) e pausa (5 min)
- Notificação sonora ao fim de cada ciclo
- Histórico de sessões salvo no LocalStorage
- Troca automática entre modos

## Tecnologias

- HTML5
- CSS3
- JavaScript (Vanilla)
- Web Audio API
- LocalStorage

## Como usar

Clone o repositório e abra o `index.html` no navegador:

git clone https://github.com/PedroNogueira2009/pomodoro-timer.git

Ou acesse diretamente pelo GitHub Pages.
